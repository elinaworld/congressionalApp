# App for Congressional App Challenge

For this challenge, we will be creating an app that is able to scan and sort various different forms of trash in order to better the environment. Users will also be able to learn more about sorting different household trash and be able to earn points from taking quizzes. 

## How To

Login / Sign-Up Function: Users are able to create their own account and username to sign up.
Information / Quizzes Function: Users are able to learn more about classifying garbage and take quizzes to earn points.
Scoreboard Function: Users are ranked based on their points from quizzes.
Map Function: Users can enter their ZIP code and get an interactive map showing local recycling centers.
Garbage Classification Function: Users take a photo of their garbage and then the app will show what type of trash it is classified as and the confidence level of the model.
